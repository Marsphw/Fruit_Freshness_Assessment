You need to download the raw dataset from the following link: https://data.mendeley.com/datasets/xkbjx8959c/2. If you'd like to use other datasets, you need to arrange the images with this kind of structure in this folder:
```
├── fruit_name_1
│   ├── condition1
│   │   ├── image1.jpg
│   │   ├── image2.jpg
│   │   ├──...
│   ├── condition2
|   |   ├── image1.jpg
|   |   ├── image2.jpg
|   |   ├──...
│   ├──...
├── fruit_name_2
│   ├── condition1
│   │   ├── image1.jpg
│   │   ├── image2.jpg
│   │   ├──...
│   ├── condition2
|   |   ├── image1.jpg
|   |   ├── image2.jpg
|   |   ├──...
│   ├──...
├── ...
```
